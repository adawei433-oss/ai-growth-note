# 成长手记：把 AI 用进真实工作

这是一个关于如何高效使用 AI 工具的分享网站。

## 📖 内容概览

本网站展示了三个 AI 使用阶段：

1. **问答模式** - 复杂问题拆解与内容生产
2. **Skill 固化** - 标准化交付物批量生成  
3. **Vibe Coding** - 把个人想法做成可用工具

## 🚀 快速开始

- 打开 `index.html` 文件即可查看
- 支持鼠标滚动、键盘方向键、触屏滑动等多种导航方式

## ✨ 特点

- 响应式设计，适配各种屏幕尺寸
- 平滑的页面过渡动画
- 交互式导航点和翻页按钮
- 图片灯箱放大效果
