02 · Skill — 绩效健康度

场景非常多，eg：周报、个人定制报告等。以绩效健康度为例，点击图片可放大查看。

03 · Vibe Coding — 本地智能检索
面对 13G 超额材料
工具：Codex × Python。部署在本地 / 办公室内网的智能问答小程序。点击图片可放大。
核心特性
  • 语义搜索：不限关键词，理解意思即可检索
  • 秒级响应：本地部署，无网络延迟
  • 多格式支持：文本、表格、图像、视频全支持
  • 知识补充：自动化爬虫更新 + 手工补充
📚
已上线运营中
使用场景
  • 会议同步查阅：实时搜索最新政策、数据
  • 报告快速成文：复用往年框架和数据
  • 新人快速上手：减少重复提问和培训时间
  • 团队知识沉淀：形成可迭代的「活文档」
